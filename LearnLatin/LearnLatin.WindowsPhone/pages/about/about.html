about

The Latin Reader is a free reading app available for anyone who wants to improve their Latin skills. In this version, it is assumed that you already have some background in the Latin language to use this app, as no grammar guidance is provided. A grammer lookup is planned for a future update.

The code for this app is provided on GitHub, an online source code repository. You can log bugs, ask for changes, or suggest new features by logging an issue on the GitHub site.

Texts in this app have been provided by the Perseus project through Tufts.edu. These texts are available under the the Creative Commons ShareAlike 3.0 license.
