About Latin Reader

The Latin Reader is a free, open-source app available for anyone who wants to brush up on their Latin skills. As of the current version, it is assumed that you already have some background in the Latin language to use this app (no grammar guidance is provided). A grammer lookup is planned for a future update.

The code for this app is provided on GitHub, an online source code repository.
You can log bugs, ask for changes, or suggest new features by logging an issue on the GitHub site.

Texts in this app have been provided by the Perseus project through Tufts.edu. These texts are available under the the Creative Commons ShareAlike 3.0 license.
